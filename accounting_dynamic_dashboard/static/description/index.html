Accounting Dynamic Dashboard

Premium UI & Merged Balances

Developed by: Imran Hoque

This module completely revamps the standard Odoo Accounting Dashboard. It introduces a highly dynamic, beautifully styled Kanban interface with custom animations, soft shadows, and clean gradients for the best user experience.

Key Features

  • ✅ Premium Aesthetics: Modern CSS styling applied to Kanban cards including soft box-shadows, rounded corners, and a clean background.
  • ✅ Dynamic Animations: Smooth hover micro-animations that gently lift the cards upon mouse interaction, creating an engaging user experience.
  • ✅ Visual Data: Beautiful linear-gradient backgrounds added to the graph sections to make data visualization stand out.
  • ✅ Clean Workspace: Automatically removes unnecessary Odoo breadcrumb text (like the duplicate "Dashboard" title next to the search bar) via targeted CSS.
  • ✅ Intelligent Routing: Overrides the default accounting menu to automatically display this custom dashboard upon entering the Accounting application.
  • ✅ Custom Controls: Adds custom fields to the journal settings, allowing strict control over which journals appear.
